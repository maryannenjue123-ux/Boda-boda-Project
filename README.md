# Boda-boda Project

The Boda-Boda Financial Management System is a PHP-MySQL web application designed to digitize member registration, payment tracking, reporting, and SMS communication for boda-boda stages, replacing manual record-keeping with an automated, secure, and scalable system.

Key Features:

Secure admin login and dashboard

Members management (add, edit, delete, view)

Payments/transactions tracking

SMS alerts via Africa's Talking

Structured database (Users, Members, Payments, Reports)

How to reach hosting:

Designed DFD, ERD, and system architecture

Built front-end pages and PHP back-end

Created and connected the MySQL database

Integrated SMS API and tested on XAMPP (localhost)
